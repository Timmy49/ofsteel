NOTE: This tool is for educational purposes only. The developer holds no responsibility for misuse.

Info

OFSteel is a Tampermonkey script that logs JSON-RPC data from OFS and stores it in Firebase or allows local downloads. This documentation outlines its features, installation steps, and usage.

Installation

To install OFSteel, first install Tampermonkey on your browser. Then, visit the home page and click "Install OFSteel" to add the script.

NOTE: Developer mode needs to be enabled in Chrome extensions in order for Tampermonkey to inject scripts. Open chrome://extensions and enable developer mode in the top right of the screen.

Usage

Once installed, the script automatically tracks login data on OFS and saves it to Firebase. You can also download the data for offline use.
