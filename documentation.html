OFSteel Documentation

Introduction

OFSteel is a Tampermonkey script designed to log JSON-RPC data from OFS and store it in Firebase or download it locally. This documentation provides an overview of its features, installation instructions, and usage guidelines.

Installation

To install OFSteel, you need to have Tampermonkey installed in your browser. Once Tampermonkey is set up, you can install the OFSteel script by clicking the "Install OFSteel" button on the home page.

Usage

After installation, the script will automatically track login information on OFS and store it in Firebase. You can also download the data locally for offline access.
